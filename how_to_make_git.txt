1. Создаем репозиторий в гитхабе (или другом удаленном репозитории)
2. Заходим в нашу локальную папку (/home/cate/some_events к примеру)
3. Делаем git init
4. git checkout -b <название ветки>
5. Создаем .gitignore файлик и добавляем туда .git/ и .idea/ и любые остальные файлы которые нужно только вам в локальной разработке
6. git add .
