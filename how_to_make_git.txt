1. Создаем репозиторий в гитхабе (или другом удаленном репозитории)
2. Заходим в нашу локальную папку (/home/cate/some_events к примеру)
3. Делаем git init
4. git config --global user.email "<имейл>"
5. git config --global user.name "<имя>"
6. git checkout -b <название ветки>
7. Создаем .gitignore файлик и добавляем туда .git/ и .idea/ и любые остальные файлы которые нужно только вам в локальной разработке
8. git add .
9. git commit -m "<краткое сообщение что вы сделали>"
